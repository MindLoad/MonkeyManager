# MonkeyManager
Application to keep your secrets safely

![ScreenShot](ScreenShots/Screen.jpg?raw=true)

## Hot keys:

- Double click by PASSWORD cell to see password in table.
- **F4**: edit field (only if row is selected).
- **DEL**: delete selected row.
